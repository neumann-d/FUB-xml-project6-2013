==== SPARQL Queries ====
für 3. alle Produkte die "ISBN" als productID enthalten:  
SELECT * WHERE {
  GRAPH ?g { ?s <http://www.w3.org/1999/xhtml/microdata#http://schema.org/Product%23:productID> ?isbn FILTER regex(?isbn, "ISBN", "i") . }
}
LIMIT 100
